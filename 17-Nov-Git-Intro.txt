Git Commands
------------
-> GIT Commands we are going to use to perform operations in git repository

git init : It is used to initialize our folder as git folder

git add : To add files to staging area (making files eligible for commit)

git add remote : to add remote repo url

git commit : to commit files to local repo

git push : to push files from local to central repo
